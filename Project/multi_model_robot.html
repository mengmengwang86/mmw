Multi-model Interactive Robot

Project Overview

This project focuses on developing a multi-model active interaction face robot at the Creative Machines Lab, Columbia University. The robot integrates cameras, microphones, a speaker, an eyeball module, and motors to achieve human-like facial expressions and interactions.

The software system leverages LLM technology, utilizing GPT-4 for natural language processing, enabling advanced video analysis and interactive capabilities. This setup allows for more lifelike and responsive communication with users.

Key Features

  • Advanced NLP integration with GPT-4
  • Combination of multiple sensory inputs (cameras, microphones)
  • Motorized eyeball and facial movement for realistic expressions
  • Interactive sound output through a speaker system
